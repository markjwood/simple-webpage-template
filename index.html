About the Template

This is nothing special. If you know even the basics of HTML and CSS, you could write something like it yourself. It's so simple, in fact, that while learning to code, I'd write something like it from scratch, every time I wanted to try out some ideas.

What I realized, though, was that I was wasting more time than I'd care to admit trying to make my experimental project look nice, even though it was never intended for public consumption. So, I finally decided to just save this as a template that I can re-use and stop wasting time on things that aren't important.

That said, maybe a simple template like this would be useful to other people. Maybe someone will even use it as the basis for a larger project. Probably not, but it's possible. Or, maybe someone who's learning, who procrastinates like I do, would appreciate having a pre-made template for their coding experiments.

So go ahead and use it however you'd like. Change anything you want. If it's for your own personal use, I don't care what you do with it. If it'll be publicly accessible, all I ask is that you delete my written content (meaning, the part you're reading now) and any references to my name, unless you really want to give me credit for whatever reason. If you absolutely feel like it, I wouldn't complain if you included a link to my github profile somewhere, but I don't expect it.

If you create something really great with it, I'd love to see it. You could tweet me a link, if you feel like it. Again, I'm not looking for credit. I just think it'd be cool to see that someone else used something I did to make something awesome.
